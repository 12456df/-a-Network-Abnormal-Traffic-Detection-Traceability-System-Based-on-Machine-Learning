================================================================================
  docs/ — 说明类文本文档（与根目录 pip 依赖文件分开管理）
================================================================================

本目录存放毕设与工程说明性质的 .txt。运行代码不依赖本目录；写论文、配环境时可查阅。

  项目介绍.txt
      仓库总览：主流程脚本顺序、目录树、根目录与 tools 模块职责、输入输出约定、扩展阅读。

  毕业论文编写建议.txt
      撰写顺序、各章要点与代码对应、开题对齐与附录答辩提示。

  PCAP_MANUAL_SETUP_zh.txt
      PCAP / CICFlowMeter（方案 A）在 Windows 上的手动安装：Python 3.12、Npcap、
      tcpdump、虚拟环境、patch 脚本等。

  毕业设计软硬件配置.txt
      实验机软硬件配置建议（CPU/内存/GPU、Python 与主要库等）。

  实际采用的研究方案-需求分析与技术路线.txt
      对照开题「拟采取的研究方案」，写明本仓库实际实现范围与技术路线。

根目录仍保留（pip 惯例，勿移动）：
  requirements.txt        主流程 Python 依赖
  requirements-pcap.txt     PCAP/CICFlowMeter 专用虚拟环境依赖

从项目根目录引用本文档时路径写作：docs/文件名.txt
